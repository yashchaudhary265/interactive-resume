Projects

  • INVEST

    INVEST: Bridging the Gap Between Entrepreneurs and Investors

    INVEST is a full-stack web platform built using the MERN stack (MongoDB, Express.js, React.js, and Node.js), designed to connect aspiring entrepreneurs with potential investors. It enables users to submit business ideas, showcase investor profiles, and explore a unified system for tracking all submissions in real time.

    Key Features:

    1. Seamless Dual User Form Submissions
    Entrepreneurs can submit startup ideas, funding needs, and contact info. Investors can register their investment capacity and sector interests via intuitive forms.

    2. Centralized MongoDB Storage
    All submissions are stored securely in MongoDB Atlas. Each user type (entrepreneur, investor) has a dedicated schema for validation and structure.

    3. Data Visualization Add-on
    Includes bar graphs, pie charts, radar charts, and interactive 3D visualizations using Chart.js/D3.js for top 1000 Indian stocks. Adds powerful insight tools for investment trend analysis.

    Tech Stack:

    Frontend: React.js, Custom CSS, Particles.js
    Backend: Node.js, Express.js
    Database: MongoDB Atlas
    Deployment: Vercel (frontend) + Render (backend)
    Visualization: Chart.js, D3.js, Three.js
    Form Processing: Axios, REST APIs

    Authentication & Security:
    Secure login/register system using bcrypt for password hashing. Role-based access can be added. Input validations are enforced throughout the backend to prevent abuse.

    Deployment:
    Frontend is deployed via Vercel for global performance, while backend APIs are hosted using Render. The frontend consumes backend APIs directly with absolute routes. MongoDB Atlas handles real-time data updates.

    Why It Matters:
    Startup ecosystems lack streamlined platforms for investor-founder matching. INVEST bridges this gap by centralizing idea submission, investor interest, and interactive analytics. It improves discoverability, saves time, and encourages data-driven decisions.

    Future Enhancements:
    - Add real-time chat between users
    - Enable download/export of submissions as PDF
    - AI-powered pitch evaluation via Gemini API
    - Matchmaking engine to align investors and entrepreneurs by sector

  • SOIL FARMING AGENT

    Soil Farming Agent: Smart Soil Intelligence Portal for Farmers

    Soil Farming Agent is a full-stack MERN (MongoDB, Express.js, React.js, Node.js) web application designed to empower farmers and agricultural stakeholders by providing soil-specific guidance, interactive globe-based navigation, and region-wise farming recommendations. It combines geospatial interactivity with curated soil data and AI-powered user flow.

    Key Features:

    1. Interactive Earth Globe Interface
    Users can rotate, zoom, and click on any region of the globe to access local soil types. A popup provides real-time guidance with a ‘Learn More’ button that navigates to detailed pages for each soil category.

    2. Predefined Soil Info Pages
    Fixed routes like /soil-info/clay, /soil-info/silty, etc., offer curated content on best crops, ideal weather, recommended pesticides, and farming tips tailored to each soil type.

    3. Country & State Selection with Zoom
    Dropdown options allow users to select a country and state, upon which the globe zooms into the selected region for deeper contextual engagement.

    4. Admin Dashboard & Dynamic Posts
    Role-based admin users can log in and create/update soil-related posts or content dynamically. All changes reflect in real-time on the user interface.

    5. Authentication & User Management
    Secure user and admin login with encrypted password handling (bcrypt). Supports role-specific routing and protected access.

    Tech Stack:

    Frontend: React.js, Globe.gl, Custom CSS, Earth3D, Three.js
    Backend: Node.js, Express.js
    Database: MongoDB Atlas
    Deployment: Vercel (frontend) + Render (backend)
    Routing: React Router, RESTful API Integration

    Data Management:
    Soil types, admin content, and user data are stored in MongoDB Atlas collections. Dynamic content can be created and displayed via custom admin dashboard modules.

    Deployment:
    Frontend deployed via Vercel for optimal UI performance. Backend hosted on Render for handling API requests and admin operations. CORS configured for cross-origin communication between both platforms.

    Why It Matters:
    Farmers and agri-entrepreneurs often lack digital access to localized soil intelligence. Soil Farming Agent simplifies this by offering a globe-driven experience that transforms soil data into visual, usable insights—boosting awareness, efficiency, and crop success.

    Future Enhancements:
    - Add satellite imagery overlay for selected regions
    - AI suggestions based on crop and climate data
    - Add farmer discussion forums
    - Real-time chatbot to answer farming-related queries
    - Soil testing report uploads and auto-analysis
