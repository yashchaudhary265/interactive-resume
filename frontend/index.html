EXPLORE PROJECT

🌍 Soil Farming Agent – Global Soil Info App with Admin Portal

Frontend (Client)
Built with: React.js, Custom CSS, Three.js, Country-State Picker, EarthGlobe Visualization

• Developed an animated 3D Earth interface using `react-globe.gl` and `Three.js`, allowing users to click and explore soil regions.
• Integrated dropdown menus to select countries and states, auto-zooming the globe to that region.
• Designed unique animated pages for six soil types (Clay, Loamy, Chalky, Silty, Peaty, Sandy Loam), each styled with background videos and farming content.
• Implemented role-based routing with user and admin dashboards.
• Styled using custom CSS and effects like glow animations, page transitions, and thematic visuals per soil type.

🖥️ Backend (Server)
Built with: Node.js, Express.js

• Created RESTful APIs for user registration, login, and admin post submission.
• Role-based logic for user and admin access.
• Used `bcrypt` for secure password hashing and Express middleware for JSON parsing and route logging.
• CORS configured to allow frontend communication with secure Render-hosted backend APIs.
• CRUD Operations: Complete Create, Read, Update, Delete functionality for soil data management.

🗃️ Database & Storage
Technology: MongoDB Atlas (Cloud-hosted NoSQL Database)

• Stored user data, admin posts, and soil information using structured Mongoose schemas.
• Enabled real-time updates and scalability using MongoDB Atlas cloud infrastructure.

🚀 Deployment
• Frontend: Hosted on Vercel for fast, global CDN-based rendering.
• Backend: Deployed on Render, exposing `/api/auth` and `/api/admin` endpoints.
• Environment variables securely stored in `.env` files and not exposed in public repositories.
• Used absolute URL-based API calls with `REACT_APP_API_URL` to connect client and server securely.

📊 Key Functional Features
• Interactive 3D globe: click or select region to view soil info.
• Rich content for each soil type: best crops, pesticides, weather, tips.
• Admin dashboard for adding/modifying soil info posts dynamically.
• User-friendly UI with animated visuals and dark-themed layout.
• Fully responsive across devices and platforms.
